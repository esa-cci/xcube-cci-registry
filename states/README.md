# State Artifacts

This directory contains xcube-cci state artifacts generated by the metadata
builder.

Expected generated files:

```text
states/
  dataset_states.json
  datatree_states.json
  geodataframe_states.json
  vectordatacube_states.json
```

The files in this directory apply to the `xcube-cci` version recorded in
`../build_info.json`. Older `xcube-cci` versions are supported by older
`xcube-cci-registry` releases, not by parallel version directories.

The initial state format should remain compatible with existing `xcube-cci`
package data. The builder must preserve manually curated fields from previous
state files:

- `places`
- `var_names`
- `pattern`

Additional build metadata belongs in `build_info.json`, not inside the legacy
state files unless `xcube-cci` is updated to support a new schema.
