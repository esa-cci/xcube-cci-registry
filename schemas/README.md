# Schemas

This directory contains JSON Schemas for validating generated artifacts.

Schemas define the contract between:

- the metadata builder;
- the registry repository;
- `xcube-cci`;
- `esa-climate-toolbox`;
- other consumers.

Initial schemas are intentionally permissive while the artifact model is being
stabilized.

Use `build_info.schema.json` for `../build_info.json`.
