# Reports

This directory contains human-readable reports generated by the metadata
builder.

Reports are for review and diagnostics. They are not the primary machine
interface for `xcube-cci` or `esa-climate-toolbox`.

Useful reports include:

- changed states;
- new or removed data IDs;
- failed describe/open checks;
- preserved manual fields;
- manual fields attached to removed data IDs;
- registry entries needing curation.
